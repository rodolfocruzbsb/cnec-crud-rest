# cnec-crud-rest
Aula ministrada na Faculdade CNEC Unaí

Projeto de CRUD utilizando REST e Spring Boot, para que os alunos possam criar Clients baseado nestes serviços.
#Deploy
Projeto com Deploy realizado no Heroku. URL: https://cnec-crud-rest.herokuapp.com/api/alunos

#API

## Buscar todos
- Método: GET
- Path: /

## Carga de dados
- Método: GET
- Path: /carga/{prefixo}
